   session
========
session使用redis保存数据,因此需要配置好redis

* 配置
config/share/app.php
```
   "session"=>[
        "name"=>"TSESSIONID",
        "cache_expire"=>60*60*2,
        "path"=>"/",
        "domain"=>"",
        "secure"=>false,
        "httponly"=>true,
    ],
```
* 使用session()函数
```
    session()->get($key);
    session()->set($key, $value);
    session()->del($key);
    session()->clear();
```
