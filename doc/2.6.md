   Fis
========
#####开启fis
1. 服务器根目录需要有  fis-conf.js, package.json 文件, 这2个文件里面的内容参考[trendi/boilerplate](https://github.com/trendi/boilerplate)
2. 安装 [nodejs](https://nodejs.org/en/), [npm](https://www.npmjs.com/), 安装教程请看各自官网

```
//安装fis
npm install -g fis3
//安装必要依赖
npm install
//启动服务器
sudo ./trendi server:restart
```
