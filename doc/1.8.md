   pool server&&连接池
========
* 启用pdo数据库连接池
    1. 打开config/share/storage.php
    修改

    ```
    "pdo"=>[
        "adapter"=>"default",//pool or default

     ```
     为

     ```

  "pdo"=>[
        "adapter"=>"pool",//pool or default

     ```
    2.  配置redis,pdo服务器配置

    ```

        "pdo"=>[
             "adapter"=>"pool",//pool or default
            "type" => "mysql",
            "prefix" => "putao_",
            "master" =>[
                "host" => "",
                "user" => "root",
                "port" => "3306",
                "password" => "",
                "db_name" => "",
                "timeout"=>5,
            ]
        ],
        "redis"=>[
             "adapter"=>"pool",//pool or default
            "servers"=>[
    //            "tcp://127.0.0.1:6379",
            ],
            "options"=>[
                'prefix'  => 'test',
                'cluster' => 'redis',
                "timeout"=>9,
            ],
        ]

    ```
* 目前支持redis 和pdo 连接池
